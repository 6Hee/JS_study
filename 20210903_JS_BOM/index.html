자바스크립트 - Browser Object Model

브라우저의 문서 공간을 제외한 모든 영역

window 객체 : 웹에서 모든 객체를 포함하는 객체

window.document.write("출력내용") -> document.write("출력내용")

window > BOM || DOM || javascript 내장 객체(날짜객체, 수학객체, 문자객체, 배열객체, Object)

BOM - history, navigator, location, screen, 팝업창 형태(alert, prompt, confirm), setInterval, setTimeout

window.open() : 새로운 창을 열 때 사용(예시, 윈도우 팝업창 : 기본 페이지 -> 새로운 문서를 연다.)

setInterval : 일정시간을 간격으로 함수 내의 실행문을 실행하는 메서드. 함수문(function) + 실행문 + 대기시간

0부터 올라가는 숫자 : 000

10부터 내려가는 숫자 : 10

이미지 슬라이드(이미지 교체)

setTimeout : 지정된 시간이 지난 후 한번만 실행하고 종료한다. (함수문 + 실행문 + 시간)

배경색 한번만 바꾸기

screen 객체 : 현재 모니터의 (실효)너비와 (실효)높이를 가져올 수 있음(모바일과 태블릿 공간은 브라우저가 화면 전체에 띄워지기 때문)

location.href : URL 주소창의 내용을 변경(a태그를 사용할 수 없는 경우, 태그 내부에 작성하여 사용 가능)
