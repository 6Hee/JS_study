    실습1. 이미지 넣기

    세장의 이미지가 모두 들어가면 버튼을 없앤다.

    힌트 : 가상의 수를 지정한 후 한장씩 내부에 넣을 때마다 가상의 수를 하나씩 올린다. 최종적으로 3번 올라가면 더 이상 올라가지 못하도록 막고 버튼은 보이지 않도록 처리한다.

    실습2. 객체 이동시키기

    이전/다음 버튼 클릭시 해당하는 이미지 순서대로 맨좌측에 보여주기(클릭시마다 first()/last()를 지정하여 prepend 또는 append를 사용하여 객체를 이동시킬 것)

    컨텐츠 페이지 구성하기

    자동 회전 슬라이드(좌측으로 이동하는 슬라이드)

    애니메이션 효과부여 $("선택자").animate({스타일 선언명 : 스타일 선언값}, 애니메이션 작동 시간, function(){});

    애니메이션 작동시간 : 밀리세컨드 단위(1초 = 1000밀리세컨드)

    function(){ 실행문 } : 콜백함수 => 애니메이션 작동(스타일 적용)이 종료된 후 함수를 실행하라

    이미지가 우측으로 이동한 다음에 이동이 종료되면 테두리 색상을 부여
