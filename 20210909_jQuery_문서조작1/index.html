문서 객체 조작

$("선택자").html() : 선택자가 하위에 포함하고 있는 모든 요소 및 텍스트를 저장 - get 방식(변수가필요)

$("선택자").html("새로운 요소 및 텍스트" 또는 구조를 구성하고 있는 변수) : 선택자의 하위요소에 "새로운 요소 및 텍스트" 또는 구조를 구성하고 있는 변수로 변경(갱신-renewal) - set 방식

Special Dishes

$("선택자").attr("속성명") : 선택자 태그 내부에 위치한 속성의 속성값을 가져온다. - get 방식(변수가 필요)

$("선택자").attr("속성명", "새로운 속성값") : 선택자 태그 내부에 위치한 속성의 속성값을 새로운 속성값으로 교체, 생성, 삭제 - set 방식

removeAttr("속성명") : 기존 태그 내부에 존재하던 속성을 완전히 제거. 속성명이 제거됨으로써 속성값도 한꺼번에 제거됨

removeClass("제거해야할 클래스명") : 기존에 존재하던 class의 속성값 중에서 지목된 클래스명만 제거. class="box active" = removeClass("active") => class="box"
