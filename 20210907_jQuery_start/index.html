형제 요소 선택자 -

prev() : 기준으로부터 바로 직전 형제만 선택

prevAll() : 기준으로부터 이전 형제 모두 선택(기준인 본인은 제외)

prevUntil("선택자") : 기준으로부터 이전 형제를 선택하는 과정에서 선택자 형제의 다음까지의 형제들을 선택한다.

next() : 기준으로부터 바로 다음 형제만 선택

nextAll() : 기준으로부터 이후 형제 모두 선택(기준인 본인은 제외)

nextUntil("선택자") : 기준으로부터 이후 형제를 선택하는 과정에서 선택자 형제의 이전까지의 형제들을 선택한다.

siblings() : 기준으로부터 형제를 모두 선택(기준인 본인은 제외)

siblings("선택자") : 형제를 찾는 과정에서 선택자에 해당하는 형제만 찾는다.
