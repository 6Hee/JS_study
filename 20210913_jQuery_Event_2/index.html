클릭한 곳의 결과값 넣기

클릭 이벤트 주의 사항 - a 태그 내부에 href 속성에 의한 페이지 새로고침 현상을 막아야 하는 경우

1. 스크립트 코드 내부에서 막는 방법 1 : 클릭 이벤트 종료 직전에 return false; 라는 문구를 작성

2. 스크립트 코드 내부에서 막는 방법 2 : 클릭 이벤트에 의한 내부 문구를 실행하기 직전에 매개변수 (e 또는 event)를 설정한 후, e.preventDefault(); | event.preventDefault(); 라고 작성

3. HTML 구문상에서 클릭할 요소의 href 속성값으로 내장 스크립트로 막는다. href= "javascript:void(0);" 작성

마우스 관련 이벤트 : mouseover, mouseout, hover(mouseenter, mouseleave), mousemove

hover 이벤트를 활용한 상단 헤더 구성하기(주메뉴, 서브메뉴) + slideDown()/slideUp() 효과 메서드
